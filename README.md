# przetrwac-lacznosc
Projekt Przetrwać "Łączność"

Grupa docelowa: uczniowie klas 8 szkół podstawowych w okresie wybierania szkoły ponadpodstawowej i uczniowie klas 1 Zespołu Szkół Łączności.

Cel: przekaz młodym adeptom Zespołu Szkół Łączności im. Obrońców Poczty Polskiej w Gdańsku jak opanować cały obszerny materiał obowiązujący w technikum 5-letnim.

Zawartość: poradniki, kursy i kompendium wiedzy.

Frontend - Next.js

Backend - Node.js + Express

Server - Vercel

Mobil app - React Native
